 ### Technologies
 This project was built using:
 - React.js
 - Node.js
 - VSC 
 - React Router
 - React Bootstrap
 
 It is hosted on RaspberryPi Zero 2W running Ubuntu Server 20.04.<br/>
 CI/CD Pipeline created with Github Actions Runner(self-hosted), Docker and Gated Commits!
